### Birb

Help Birb get to her friends before dusk.

Use arrows to navigate Birb through the maze and get to her friends.

This game was written during [Spring Lisp Game Jam 2021](https://itch.io/jam/spring-lisp-game-jam-2021 "Spring Lisp Game Jam 2021") with [Chickadee](https://dthompson.us/projects/chickadee.html "Chickadee") game development toolkit for Guile Scheme.

The pixel arts are mine and the music is by [Rick Hoppmann (TinyWorlds)](https://opengameart.org/users/tinyworlds "Rick Hoppmann (TinyWorlds)").
